# 需求规格说明书

## 1. 目标

系统用于集中管理百度网盘分享链接，浏览分享目录，并将指定远端目录增量同步到
Windows、Linux 或 macOS 本机目录及操作系统挂载的 SMB NAS。

## 2. 用户与权限

- 用户由 `data/password.txt` 维护，格式为 `username:password:role`。
- 角色包含管理员 `admin` 和普通用户 `user`。
- 管理员拥有全部页面和操作权限。
- 普通用户可分别获得分享链接、同步映射、任务中心、系统配置页面权限。
- 所有权限必须由后端校验，不能只隐藏前端导航。

## 3. 功能需求

1. 登录、退出和七天会话。
2. 分享链接的新增、编辑、删除、刷新和逐级目录浏览。
3. 同步映射的新增、编辑、删除、连接检测、单项同步和全部同步。
4. 支持本机路径和操作系统挂载的 SMB 路径。
5. 支持手动同步和按映射配置的分钟间隔自动同步。
6. 任务中心显示实时状态、消息和持久化同步历史。
7. 系统配置页显示运行配置并维护百度网盘 Cookie。
8. 权限管理页维护用户密码、角色和页面权限。

## 4. 非功能需求

- 后端使用 Python、FastAPI、SQLite；前端使用 Vue 3。
- 支持 Windows、Linux、macOS。
- 密码以 scrypt 哈希存入数据库，Cookie 与会话密钥不返回浏览器。
- 时间按北京时间显示，日志按天轮转。
- SMB 掉线时禁止向失效挂载点下的本机目录写入文件。
