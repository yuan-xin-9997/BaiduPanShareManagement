# 设计说明书

## 1. 目录与组件

- `src/app/bdpan`：FastAPI、百度网盘客户端、SQLite、同步与后台任务。
- `src/app/frontend`：Vue 3 + Vite 源码。
- `src/app/bdpan/web_static`：Vue 构建产物，由 FastAPI 提供。
- `config/app.json`：非敏感运行配置。
- `data/app.sqlite3`：业务数据库。
- `data/password.txt`：账号来源。
- `data/secrets.json`：自动生成的会话密钥和百度 Cookie。
- `logs`：轮转日志、PID 与进程输出。
- `src/JenkinsConfig`：Jenkins Pipeline 定义。

## 2. 认证与授权

启动和登录时解析 `password.txt`，将用户密码转换为带随机盐的 scrypt 哈希并同步到
SQLite。浏览器保存签名、HttpOnly、SameSite=Lax 会话 Cookie。每个受保护 API 根据
当前用户角色和 `user_page_permissions` 表执行授权。

## 3. 后台任务

耗时的分享索引和同步任务在线程池执行。调度线程按配置间隔扫描启用自动同步的映射，
同一映射禁止并发执行。同步历史写入 `sync_runs`。

## 4. 存储安全

Windows SMB 使用 UNC 或远程驱动器；Linux/macOS 要求路径位于真实挂载点之下。
每次同步前验证现存父目录和写权限，再使用同目录临时文件下载、校验大小和原子替换。

## 5. 数据迁移

首次使用新结构且 `data/app.sqlite3` 不存在时，可复制旧默认数据库。数据库启动时通过
幂等建表和 `ALTER TABLE` 增加新字段，保留分享、索引和同步映射。
